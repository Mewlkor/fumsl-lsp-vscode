# fumsl-lsp-vscode

Demo for highlighting: `fumsl` and `fumsl-lsp` in VSCode.
![](https://github.com/Mewlkor/fumsl-lsp-vscode/blob/main/vscode/lsplogs/fumsl_lsp_demo.png "fumsl-lsp demo")

## Installation
1) Open with VSCode
2) cd vscode
3) npm install
4) npm run esbuild
5) vsce package
6) Install the generated .vsix file in VSCode
